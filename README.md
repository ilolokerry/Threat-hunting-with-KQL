# Threat Hunting with KQL – KC7 Labs

---

# Project Overview

This project focuses on threat hunting using Kusto Query Language (KQL) within the KC7 platform. The lab is designed to simulate real-world security investigation scenarios where telemetry data is analyzed to detect malicious activity, uncover hidden threats, and identify attacker behavior patterns.

The exercises involved writing and executing KQL queries to investigate security logs and detect suspicious activity across different attack scenarios.

---

#  Objectives

- Develop practical threat hunting skills using KQL
- Analyze security telemetry data for suspicious behavior
- Detect attacker techniques and malicious patterns
- Improve log analysis and investigative thinking
- Simulate real-world SOC threat hunting workflows

---

#  Platform Used

- KC7 Cybersecurity Threat Hunting Labs  
- Kusto Query Language (KQL) for data analysis and investigation  

---

# Skills Demonstrated

- Threat hunting using KQL queries  
- Security log analysis and correlation  
- Identifying suspicious user and system activity  
- Investigating attacker behavior patterns  
- Interpreting security telemetry data  
- SOC-style analytical thinking and investigation workflows  

---

#  Lab Summary

Completed multiple threat hunting scenarios on KC7 by analyzing security datasets using KQL. Each scenario required identifying indicators of compromise, tracing attacker activity, and building queries to isolate malicious behavior within large datasets.

---

#  Key Learning Outcomes

- Improved understanding of structured query language for security data analysis  
- Ability to identify suspicious patterns in logs  
- Hands-on experience with threat hunting methodologies  
- Strengthened SOC analytical and investigative skills  

---

#  Conclusion

This project provided practical experience in threat hunting using KQL, focusing on real-world SOC-style investigations. It improved my ability to analyze security data, detect anomalies, and investigate potential threats effectively.
